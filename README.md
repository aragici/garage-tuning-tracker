# Garage Tuning Tracker 🔧

A web-based full-stack CRUD application developed as a course project for **System Analysis and Design (SAD)**. It allows users to register, manage vehicles in their garage, and keep a log of installed tuning modifications/parts with real-time dynamic rendering.

--- 

## 🚀 Features

- **Full CRUD Support**: Create, read, and delete vehicles and structural tuning components.
- **User Isolation**: Secure JWT-based Authentication protects custom garage spaces.
- **RESTful Architecture**: Follows clean REST endpoint routing standards.
- **Single-Page Application**: Implemented strictly with Vanilla HTML5, CSS3 (Neon-Carbon UI), and asynchronous JavaScript (`fetch`). No frameworks used.
- **Interactive Documentation**: Integrated fully interactive Swagger UI for instant API exploration.
- **Robustness**: Unit-tested business logic ensuring zero runtime configuration faults.

---

## 🛠️ Tech Stack

- **Frontend**: Vanilla HTML, CSS, JavaScript (Asynchronous API architecture).
- **Backend**: Node.js, Express.js framework.
- **Data Layer**: SQLite3 (relational local file store).
- **Testing**: Jest.
- **API Spec**: Swagger UI / OpenAPI 3.0.
